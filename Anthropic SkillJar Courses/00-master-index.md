## Key Insights Across the Catalogue

### What Anthropic is teaching
The course catalogue reveals Anthropic's vision for how humans should work with Claude:
1. **Three modes** — Chat (conversation), Cowork (agentic files), Code (developer) — each for different contexts
2. **Context management is critical** — CLAUDE.md, Skills, Projects, context commands are all about managing what Claude knows across sessions and within sessions
3. **The Skills system is central** — Three courses touch Skills in different contexts; Anthropic is investing heavily in this abstraction
4. **MCP is the integration layer** — MCP appears in 5 courses; it's the standardised way to connect Claude to everything
5. **Responsible use is foundational** — The 4D Framework courses (5 variants) show Anthropic's commitment to teaching judgment, not just tool use

### The two curricula running in parallel
**Technical curriculum:** Claude Code 101 → Claude Code in Action → Agent Skills → Subagents → Building with API → MCP Intro → MCP Advanced → Bedrock/Vertex
**Human curriculum:** AI Fluency Framework → AI Capabilities & Limitations → Applied variants (Educators, Students, Nonprofits, Teaching)

Both are necessary. Most organisations invest in one and neglect the other.

### What's free vs. registration-required
**Free courses:** Claude 101, Introduction to Claude Cowork, AI Fluency: Framework & Foundations, Introduction to Agent Skills, Introduction to Subagents
**Registration required (likely free or low cost):** All others

## Master Workflow Integration Guide

This synthesises the most actionable items across all 17 courses into an implementation roadmap.

### Week 1: Foundations
- [ ] Complete Claude 101 → set up Projects for all active work areas
- [ ] Complete AI Fluency: Framework & Foundations → adopt 4D checklist for high-stakes work
- [ ] Set up CLAUDE.md in every active codebase
- [ ] Create your first 3 Skills (global)

### Week 2: Cowork & Code Setup
- [ ] Complete Introduction to Claude Cowork → set up context files and first scheduled task
- [ ] Complete Claude Code 101 → implement Explore→Plan→Code→Commit as default rhythm
- [ ] Install 3 core hooks: post-edit linter, pre-commit test runner, session log
- [ ] Create 3 subagents: code-reviewer, doc-generator, researcher

### Week 3: Customisation & Depth
- [ ] Complete Claude Code in Action → implement custom command library
- [ ] Complete Introduction to Agent Skills → build team skill library in repo
- [ ] Complete Introduction to Subagents → implement parallel analysis pattern
- [ ] Complete AI Capabilities & Limitations → apply diagnostic framework

### Week 4+: Technical Integration
- [ ] Complete Building with the Claude API → set up automated prompt eval pipeline
- [ ] Complete Introduction to MCP → build MCP server for primary data source
- [ ] Set up GitHub integration for automated PR review
- [ ] Complete MCP Advanced Topics when ready for production MCP deployment

### Ongoing
- [ ] AI Fluency: Framework & Foundations → run as team training
- [ ] Teaching AI Fluency → design internal AI fluency curriculum
- [ ] Revisit and update Skills, CLAUDE.md, and subagent definitions monthly

## Notes & Observations

- **Course quality is high** — The academic partnerships (Ringling, UCC, GivingTuesday) elevate the AI Fluency courses beyond typical corporate AI training
- **The free courses are the best starting points** — Anthropic clearly made the highest-impact onboarding courses (101, Cowork, Skills, Subagents, AI Fluency Framework) free to maximise adoption
- **Certificate programme** — Claude 101 includes a completion certificate; likely others do too — useful for team accountability and formal onboarding programmes
- **Video-based courses** — The developer courses (API, Claude Code in Action) are video-based; some videos may still be processing as of the capture date
- **Active content** — Several video pages showed "video still being processed" notices — the platform is actively being built out
- **Anthropic-AWS partnership** — The Bedrock course originated as accreditation training for AWS employees, which speaks to the depth and real-world validation of that content
