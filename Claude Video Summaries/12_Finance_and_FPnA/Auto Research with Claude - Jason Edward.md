**Premise**

As an entrepreneur, you need to be up-to-date with everything in your industry. World changes too fast. Solution: AI agents that auto-research and email you a research report every morning. Use cases: spy on competitors, content for social media, market research.

**Setup (5 Minutes)**

**Prerequisites**

-   Claude subscription (Pro or higher).

-   Claude Code OR Claude Cowork (same skill works in both).

**Step 1 --- Install the Skill**

1.  Go to Jason Edward\'s GitHub repo (link in his video description).

2.  Copy the repo URL.

3.  Paste into Claude Code/Cowork: \'Set this up.\'

4.  Claude downloads the research skill.

**Step 2 --- Get SuperData API Key (Free, 100 Scrapes/Day)**

5.  Go to SuperData website.

6.  Sign in.

7.  Top-left → API Keys → Create New API Key.

8.  Copy.

SuperData scrapes YouTube, social media, blog posts.

**Step 3 --- Get Resend API Key (Free, Sends Emails)**

9.  Go to Resend.com.

10. Create account.

11. API Keys → Create API Key.

12. Name it. Full Access. Add.

13. Copy.

**Step 4 --- Configure**

In Claude Code/Cowork, paste:

> *super data API key: \[paste\]*
>
> *resend API key: \[paste\]*
>
> *Please run the skill*

**Running the Research**

Type your topic. Example: \'email marketing for e-commerce businesses.\' Skill asks:

-   What type of information do you want? (Sub-topics suggested: AI personalization, abandoned cart, browser abandonment, post-purchase flows, deliverability).

-   What sources? (YouTube, news articles, blog posts, social media).

Choose \'all\' to maximize coverage.

**What Happens**

14. Claude launches 4 parallel research agents.

15. YouTube agent uses 63 tools, 80K tokens.

16. All agents complete in \~7 minutes.

17. Skill saves the report.

18. Email sent via Resend API.

19. Total time: \~10 minutes from prompt to email.

**Schedule It**

After first run, tell Claude:

-   \'Set this up to run every 24 hours.\'

-   \'Set this up to run every Monday at 7 AM.\'

-   Or any other cadence.

As long as your main computer is on, it executes for you. Wake up to a research report in your inbox.

**Why This Matters**

Manually doing this research would take hours. Now it\'s automated. The host\'s claim: this barely scratches the surface of what\'s possible. Once you understand AI agents, you can automate WhatsApp, Telegram, virtually any aspect of your life.
