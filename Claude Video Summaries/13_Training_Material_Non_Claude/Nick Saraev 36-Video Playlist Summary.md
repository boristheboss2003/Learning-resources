**Cross-Cutting Themes Across All Videos**

**1. The Build & Sell Flywheel**

Build for yourself first → document patterns → find client with same need → deliver customized version → refine → repeat. Same principle whether using Claude Code, n8n, or a hybrid.

**2. Layered Tool Stack**

**3. Recommended Beginner Path**

1.  Master Claude Code (or Cowork) basics. Use the 4hr or 36-min crash course.

2.  Build first product end-to-end (vibe coding course).

3.  Add n8n for cross-tool automation. Use \'Master 80% of n8n\' or 6hr course.

4.  Layer in skills, sub-agents, and Karpathy auto-research patterns.

5.  Productize one of your systems and sell it.

**4. Skills \> Generic Agents**

Saraev\'s controversial take: most \'AI agent\' demos are over-engineered. Skills (markdown SOPs) are the real durable productivity asset. Skills compound. Generic agents need re-prompting every time.

**5. Token Management Matters**

-   Use Opus 4.5+ with thinking --- counter-intuitively cheaper because uses fewer tokens overall.

-   Clear/reset context aggressively. Don\'t fill 1M context window just because you can.

-   Skills with progressive disclosure save tokens vs mega-prompts.

-   Multi-agent burns tokens 3-4x --- only when justified.

**6. Verification Is Non-Negotiable**

Common across all videos: model can be confidently wrong. Always:

-   Test in browser/run code/check formulas before shipping.

-   Audit gates before output gates.

-   Spot-check one or two values manually per output.

**7. The Productized Service Economy**

Multiple videos converge on this: AI implementations sell for \$5K-\$150K+. Tiers based on scope and complexity. Build & Sell with Claude Code (10hr) and Build & Sell n8n (8hr) are full curricula on this.
